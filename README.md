# Raspberry-Pi-Pico-HID-Keyboard-for-GTA-Cheats
A custom USB HID device built with Raspberry Pi Pico 2 W that automatically types GTA V cheat codes. Navigate through a scrollable OLED menu with a joystick and execute cheats with a single button press!
